# 만원의 행복

만원의 행복 프론트엔드 저장소

## 프로젝트 기간 : 2023.10 ~ 2023.12

## 프로젝트 소개 : 한 푼이라도 아쉬운 사람들을 위한 만원의 행복 서비스

- 착한가게 업소 정보를 제공합니다.
- 짠지도를 통해 위치기반 검색이 가능합니다.
- 짠처리를 통해 업소의 할인 상품을 판매합니다.
- 단짠단짠을 통해 커뮤니티 기능을 제공합니다.

## 기획 동기

고물가와 고금리로 인해 국민들의 생활비 부담이 증가하여 현재는 만원으로 할 수 있는 것이 매우 제한적입니다. 이러한 사회적 흐름에 발맞춰, 단돈 만원으로 저렴하게 이용할 수 있는 착한 가게 들을 알려주는 공공 서비스를 만들고자 합니다.
